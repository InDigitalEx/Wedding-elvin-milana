# 🎉 Свадебный сайт с системой приглашений

Красивый и интерактивный сайт для вашей свадьбы с полнофункциональной админ-панелью, системой уникальных приглашений и генерацией QR кодов.

## 🚀 Особенности

### 🎨 Дизайн
- Элегантное оформление в палитре бежево-коричневых тонов (шоколадно-коричневый & золотистый бежевый)
- Анимации и плавные переходы
- Параллакс эффекты
- Полностью адаптивный дизайн для всех устройств
- Красивая героическая секция с цветочными элементами

### 💼 Админ-панель
- Полное управление параметрами свадьбы
- Редактирование информации (имена жениха/невесты, дата, место, время)
- Конструктор расписания события с временной шкалой
- Управление дресс-кодом и реестром подарков
- Выбор и настройка цветовой схемы сайта в реальном времени
- Панель статистики гостей

### 💌 Система приглашений
- **Два типа приглашений:**
  - Личные приглашения для отдельных гостей с их именем
  - Общие приглашения (без указания имени)
- Возможность приглашения групп (пары, семьи и т.д.)
- Генерация уникального кода для каждого приглашения
- **Красивый QR код** для быстрого доступа к приглашению
- Ссылка приглашения с автоматической копией в буфер обмена

### 👥 Управление гостями
- Список всех гостей с фильтрацией по статусу ответа
- Сбор ответов на приглашение (подтвердил/отклонил)
- Учет диетических предпочтений (вегетарианское, веганское, без глютена и т.д.)
- Личные сообщения от гостей
- Подсчет планируемых гостей
- **Экспорт списка гостей в CSV** для использования в Excel/Google Sheets

### 📊 Главная страница сайта
- **О мероприятии** - место, время, программа дня
- **Расписание события** с красивой timeline визуализацией
- **Требования к дресс-коду**
- **Раздел подарков** с информацией о реестре
- **Система подтверждения** (RSVP) с формой ответа
- **Навигация** между разделами сайта

## 🛠️ Технологический стек

### Backend
- **Python 3.11+**
- **FastAPI** - современный веб-фреймворк
- **SQLAlchemy** - ORM для работы с БД
- **SQLite** - база данных (хранится локально)
- **Pydantic** - валидация данных
- **python-qrcode** - генерация QR кодов
- **Pillow** - обработка изображений

### Frontend
- **HTML5** - семантическая структура
- **CSS3** - современные стили с переменными и медиа-запросами
- **Vanilla JavaScript** - без зависимостей
- **Fetch API** - работа с API
- **Resp responsive дизайн**

## 🚀 Установка и запуск

### Требования
- Python 3.11+
- Node.js (опционально, если нужен локальный сервер)
- Современный браузер

### 1. Клонировать проект

```bash
git clone <url-проекта>
cd Wedding-elvin-milana
```

### 2. Установить Backend

```bash
# Перейти в папку backend
cd backend

# Создать виртуальное окружение
python -m venv venv

# Активировать виртуальное окружение
# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate

# Установить зависимости
pip install -r requirements.txt
```

### 3. Запустить Backend

```bash
# Из папки backend
python main.py
```

Backend будет доступен на `http://localhost:8000`

API документация: `http://localhost:8000/docs`

### 4. Открыть Frontend

```bash
# Вариант 1: Просто откройте файл в браузере
# Откройте frontend/index.html в браузере

# Вариант 2: Используйте локальный сервер (Python 3)
cd frontend
python -m http.server 3000
```

Сайт будет доступен на:
- Главная страница: `http://localhost:3000`
- Админ-панель: `http://localhost:3000/admin`

## 📖 Как использовать

### Для жениха и невесты (Админ-панель)

1. **Откройте админ-панель**: `http://localhost:3000/admin`

2. **Установите параметры свадьбы**:
   - Перейдите в раздел "Параметры свадьбы"
   - Укажите ваши имена, дату, время и место
   - Добавьте информацию о дресс-коде и подарках
   - Сохраните

3. **Создайте расписание**:
   - Перейдите в раздел "Расписание события"
   - Добавьте пункты расписания с временем
   - Сохраните

4. **Создайте приглашения**:
   - Перейдите в раздел "Приглашения"
   - Выберите тип приглашения (личное или общее)
   - Для личного приглашения добавьте личное сообщение
   - Нажмите "Создать приглашение"
   - Скопируйте ссылку и отправьте гостю (или покажите ему QR код)

5. **Следите за ответами**:
   - На панели управления видна статистика по гостям
   - Перейдите в "Гости" для просмотра подробного списка
   - Экспортируйте список в CSV для использования в Excel

6. **Настройте дизайн**:
   - В разделе "Дизайн" выберите цвета, которые вам нравятся
   - Изменения применятся на главной странице в реальном времени

### Для гостей

1. **Откройте приглашение**: 
   - Гость открывает ссылку из письма или сканирует QR код
   - Видит личное приглашение с его именем и информацией о свадьбе

2. **Подтвердите присутствие**:
   - Выберите "Да" или "Нет"
   - Укажите диетические предпочтения
   - Оставьте личное сообщение (опционально)
   - Отправьте ответ

3. **Полная информация**:
   - На сайте гость видит всю информацию о мероприятии
   - Расписание события, дресс-код, информацию о подарках

## 🔐 API Endpoints

### Свадьба
- `GET /api/wedding` - Получить параметры свадьбы
- `POST /api/wedding` - Создать/Обновить параметры свадьбы
- `GET /api/wedding/full` - Получить полную информацию о свадьбе
- `GET /api/statistics` - Получить статистику гостей

### Приглашения
- `GET /api/invitations` - Получить все приглашения
- `POST /api/invitations` - Создать приглашение
- `GET /api/invitations/{code}` - Получить приглашение по коду
- `DELETE /api/invitations/{code}` - Удалить приглашение
- `GET /api/invitations/{code}/qr` - Получить QR код приглашения

### Гости
- `GET /api/guests` - Получить всех гостей
- `GET /api/guests?invitation_code={code}` - Получить гостей по коду приглашения
- `POST /api/guests` - Добавить гостя
- `PUT /api/guests/{id}/confirm` - Подтвердить присутствие гостя
- `DELETE /api/guests/{id}` - Удалить гостя

### Цветовая схема

По умолчанию используются бежево-коричневые цвета:
- Основной (Primary): `#8b7355` - Шоколадно-коричневый
- Вторичный (Secondary): `#d4a574` - Золотистый бежевый
- Дополнительный (Accent): `#e8d5c4` - Светлый бежевый

Смените цвета в админ-панели в разделе "Дизайн"

Все переменные стилей находятся в `frontend/css/style.css`:

```css

    --primary: #8b7355;
    --secondary: #d4a574;
    --accent: #e8d5c4;
    /* ... и т.д. */
}
```

## 💾 Хранение данных

Все данные хранятся в локальной БД SQLite (`backend/wedding.db`):
- Параметры свадьбы
- Приглашения и их коды
- Информация о гостях
- Статусы ответов на приглашения

Нет отправки данных на сторонние серверы. Все остается на вашем компьютере!

## 🚀 Развертывание

### Развертывание на сервер

1. **Backend** (Python/FastAPI):
   - Хостинг: Heroku, PythonAnywhere, DigitalOcean, AWS EC2
   - Скопируйте папку backend на сервер
   - Установите зависимости: `pip install -r requirements.txt`
   - Запустите: `uvicorn main:app --host 0.0.0.0 --port 8000`

2. **Frontend** (HTML/CSS/JS):
   - Хостинг: GitHub Pages, Netlify, Vercel, Firebase Hosting
   - Скопируйте папку frontend на хостинг
   - Обновите `CONFIG.API_BASE_URL` в `frontend/js/config.js` на URL вашего backend
