## 📋 API Endpoints

### Свадьба
- `GET /api/wedding` - Получить параметры
- `POST /api/wedding` - Создать/Обновить
- `GET /api/wedding/full` - Полная информация
- `GET /api/statistics` - Статистика

### Приглашения
- `GET /api/invitations` - Все приглашения
- `POST /api/invitations` - Создать
- `GET /api/invitations/{code}` - По коду
- `DELETE /api/invitations/{code}` - Удалить
- `GET /api/invitations/{code}/qr` - QR код

### Гости
- `GET /api/guests` - Все гости
- `GET /api/guests?invitation_code={code}` - По приглашению
- `POST /api/guests` - Добавить
- `PUT /api/guests/{id}/confirm` - Подтвердить
- `DELETE /api/guests/{id}` - Удалить

## 🎯 Использование

### 1. Настройка свадьбы
1. Откройте админ-панель: http://localhost:3000/admin
2. Заполните "Параметры свадьбы"
3. Добавьте расписание события
4. Выберите цветовую схему
5. Сохраните

### 2. Создание приглашений
1. Перейдите в "Приглашения"
2. Выберите тип (личное/общее)
3. Заполните данные
4. Создайте приглашение
5. Скопируйте ссылку или QR код

### 3. Отправка гостям
- Отправьте ссылку в письме, мессенджере
- Или покажите QR код (можно распечатать)
- Гость откроет красивое приглашение

### 4. Отслеживание ответов
- На панели видна статистика
- В таблице гостей полный список
- Можно экспортировать в CSV
