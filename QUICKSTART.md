# 🚀 БЫСТРЫЙ СТАРТ

## За 5 минут к работающему сайту!

### Шаг 1: Запустить Backend (API + БД)

**Windows:**
```bash
run_backend.bat
```

**macOS/Linux:**
```bash
chmod +x run_backend.sh
./run_backend.sh
```

✅ Backend запущен на `http://localhost:8000`

### Шаг 2: Запустить Frontend (в отдельном терминале)

**Windows:**
```bash
run_frontend.bat
```

**macOS/Linux:**
```bash
chmod +x run_frontend.sh
./run_frontend.sh
```

✅ Frontend запущен на `http://localhost:3000`

### Шаг 3: Откройте браузер

- **Главная страница**: http://localhost:3000
- **Админ-панель**: http://localhost:3000/admin
- **API документация**: http://localhost:8000/docs

---

## 🎯 Первые шаги

### 1. Настройте свадьбу (админ-панель)

1. Откройте `http://localhost:3000/admin`
2. Перейдите в "Параметры свадьбы"
3. Установите:
   - Ваши имена
   - Дату и время
   - Место проведения
   - Дресс-код и информацию о подарках
4. Нажмите "Сохранить"

### 2. Создайте расписание

1. Перейдите в "Расписание события"
2. Добавьте пункты расписания:
   - 14:00 - Встреча гостей
   - 15:00 - Начало церемонии
   - и т.д.
3. Сохраните

### 3. Создайте приглашение

1. Перейдите в "Приглашения"
2. Выберите тип "Личное приглашение для гостя"
3. Заполните максимум гостей и личное сообщение
4. Нажмите "Создать приглашение"
5. **Скопируйте ссылку** или сохраните **QR код**
6. Отправьте гостю!

### 4. Следите за ответами

- На "Панели управления" видна статистика
- В "Гостях" полный список с ответами
- Экспортируйте в CSV при необходимости

---

## 🎨 Настройка дизайна

1. Перейдите в "Дизайн"
2. Выберите цвета:
   - Основной (коричневый): для фона и кнопок
   - Вторичный (золотистый): для акцентов
   - Дополнительный (бежевый): для светлых элементов
3. Сохраните - изменения применятся сразу на главной!

---

## 📱 Ссылка для гостей

Когда вы создаете приглашение, админ-панель дает вам:
- **Ссылка**: `http://localhost:3000/?code=INV-xxxxxxxx`
- **QR код**: Отсканируйте для открытия приглашения

Отправьте одно из них гостю - он откроет красивое приглашение!

---

## 📊 Типы приглашений

### Личное приглашение
- Использование для конкретного гостя или группы
- Может быть на 1, 2, 3+ человека
- Гость видит свое имя на приглашении
- Уникальный код для каждого

### Общее приглашение  
- Без указания имени
- Один QR код на всех
- Подходит для групп, которые ответят отдельно

## ⚠️ Важно

1. **Backend должен быть запущен** для работы API
2. **Все данные локальные** - хранятся в `backend/wedding.db`
3. **QR коды** сохраняются в папке `backend/qr_codes/`
4. **Не закрывайте** окна терминалов, пока используете сайт

## 🐛 Решение проблем

### "Ошибка подключения к API"
- ✅ Убедитесь, что Backend запущен
- ✅ Проверьте, что Backend на порту 8000

### "QR код не загружается"
- ✅ Создайте папку `backend/qr_codes/`
- ✅ Проверьте права доступа

### "Страница не загружается"
- ✅ Убедитесь, что Frontend на порту 3000
- ✅ Попробуйте обновить страницу (Ctrl+R)
- ✅ Очистите кэш браузера (Ctrl+Shift+Delete)
